Dentro de este proyecto se encuentra el análisis de los datos de la empresa Telecom X, donde se hizo énfasis en encontrar las 
razones de la evasión de clientes (churn) para poder tomar acción al respecto de esta terrible situación.
Los datos se toman de una API, por lo que no es necesario agregar nada manualmente, incluso, con solo ejecutar todas las celdas
se podrá visualizar los pasos desde el tratamiento y conversión de los datos hasta la visualización requerida para 
determinar los principales motivos de nuestro objetivo principal.
Al final del archivo se encuentra un resumen explicando los problemas encontrados en el dataset, las acciones necesarias para resolver
estos mismos y las visualizaciones en forma de gráfica de las conclusiones a las que se llegó.
